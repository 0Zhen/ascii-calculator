隱私政策

最後更新日期：2025年1月1日

本隱私政策說明 ASCII 查表工具（以下簡稱「本網站」）如何收集、使用和保護您的個人資訊。

1. 資訊收集

本網站使用 Google Analytics 來收集匿名的使用統計資訊，包括但不限於：

  • 訪問時間和頻率
  • 瀏覽器類型和版本
  • 作業系統
  • IP 地址（匿名化處理）
  • 頁面瀏覽行為

2. Cookie 使用

本網站使用以下 Cookie：

  • Google Analytics Cookie：用於分析網站流量和使用者行為
  • Google AdSense Cookie（如啟用）：用於展示個人化廣告

您可以通過瀏覽器設定來拒絕 Cookie，但這可能會影響網站的某些功能。

3. 第三方服務

本網站使用以下第三方服務：

  • Google Analytics：網站分析服務。查看 Google 隱私政策
  • Google AdSense（如啟用）：廣告服務。查看廣告政策
  • GitHub Pages：網站託管服務。查看 GitHub 隱私聲明

4. 資料使用目的

我們收集的資訊僅用於：

  • 改善網站功能和使用者體驗
  • 分析網站流量和使用模式
  • 展示相關廣告（如啟用 AdSense）

5. 資料安全

本網站採取合理的安全措施來保護您的資訊。所有透過 Google Analytics 收集的資料都經過匿名化處理。

6. 兒童隱私

本網站不會刻意收集 13 歲以下兒童的個人資訊。如果您是家長或監護人，並發現您的孩子向我們提供了個人資訊，請聯繫我們。

7. 您的權利

您有權：

  • 要求查看我們收集的關於您的資訊
  • 要求更正或刪除您的資訊
  • 選擇退出 Google Analytics 追蹤（安裝 Google Analytics 停用瀏覽器外掛程式）
  • 透過廣告設定管理個人化廣告

8. 隱私政策變更

我們可能會不時更新本隱私政策。任何變更將在本頁面上發布，並更新「最後更新日期」。

9. 聯絡我們

如果您對本隱私政策有任何疑問，請透過 GitHub 開 Issue 聯絡我們。
