# ASCII 查表工具 - 程式設計人員模式

一個功能完整的 ASCII 碼查詢與計算工具，模仿 Windows 計算機的程式設計人員模式。

## ✨ 功能特色

### 📊 多格式支援
- **CHAR 模式**: 直接按鍵盤任意鍵查詢 ASCII 碼
- **HEX 模式**: 十六進位輸入與顯示
- **DEC 模式**: 十進位輸入與顯示
- **OCT 模式**: 八進位輸入與顯示
- **BIN 模式**: 二進位輸入與顯示

### 🔢 計算機功能
支援完整的運算操作：
- 加法 (+)
- 減法 (-)
- 乘法 (×)
- 除法 (÷)
- 餘數 (%)
- 位元左移 («)
- 位元右移 (»)

### 💾 位元數模式
- **QWORD (64-bit)**: 0 ~ 0xFFFFFFFFFFFFFFFF
- **DWORD (32-bit)**: 0 ~ 0xFFFFFFFF
- **WORD (16-bit)**: 0 ~ 0xFFFF
- **BYTE (8-bit)**: 0 ~ 0xFF

### ⌨️ 鍵盤快捷鍵
- **數字/字母鍵**: 直接輸入（根據當前格式自動過濾）
- **Backspace**: 刪除最後一位
- **Escape**: 清除全部
- **Enter / =**: 執行計算
- **+, -, *, /, %**: 運算符

### 🎨 其他特色
- 暗色主題設計
- 即時格式轉換
- 點擊數值可複製
- 響應式設計（支援手機）
- 當前格式高亮顯示

## 🚀 使用方法

### 線上使用
直接訪問 GitHub Pages：`https://你的用戶名.github.io/ascii-calculator/`

### 本地使用
1. 下載 `ascii-table.html`
2. 用瀏覽器開啟即可使用

## 📖 使用範例

### ASCII 字符查詢
1. 點擊 **CHAR** 標籤
2. 按鍵盤任意鍵（例如按 `A`）
3. 立即顯示該字符的所有編碼

### 進位轉換
1. 選擇輸入格式（例如 **HEX**）
2. 輸入數值（例如 `FF`）
3. 自動顯示其他進位（DEC: 255, OCT: 377, BIN: 11111111）

### 計算範例
**例 1: ASCII 字符運算**
- 輸入 `65` (A 的 ASCII 碼)
- 按 `+`
- 輸入 `32` (空格的 ASCII 碼)
- 按 `=`
- 結果: `97` (小寫 a 的 ASCII 碼)

**例 2: 位元運算**
- 輸入 `1`
- 按 `«` (左移)
- 輸入 `3`
- 按 `=`
- 結果: `8`
